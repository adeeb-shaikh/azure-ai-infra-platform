# Observability Workbook

Workbook Name:

Azure Infra Platform - Observability Overview

## Panels

### Container App System Events by Reason

Purpose:

Provides a breakdown of Container Apps system events by event type.

### Recent Container App System Events

Purpose:

Displays the latest platform events and revision activity.

### Recent Application Logs

Purpose:

Displays recent application console output.

### Recent Failure Signals

Purpose:

Highlights known failure indicators:

- FailedMount
- ReplicaUnhealthy

Used for rapid incident triage.
